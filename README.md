# Vue Portfolio Web App

## Project Overview
This is a Vue.js-based portfolio web application designed to showcase projects, skills, and personal information in a visually appealing format.

## Folder Structure
```
.vscode/               # VS Code workspace settings
assets/               # Static assets for the project
images/               # Image files used in the application

.editorconfig         # Code style settings
.gitignore            # Files and directories to ignore in Git
.prettierrc.json      # Prettier configuration for code formatting

# Favicon and Web Manifest
android-chrome-192x192.png
android-chrome-512x512.png
apple-touch-icon.png
favicon-16x16.png
favicon-32x32.png
favicon.ico
favicon1.ico
favicon2.ico
index.html            # Main HTML file
robots.txt            # SEO and bot control settings
site.webmanifest      # Web App Manifest file
```

## Setup
To get started with this project, follow these steps:

### Install Dependencies
```sh
npm install
```

### Run Development Server
```sh
npm run serve
```

### Build for Production
```sh
npm run build
```

## Features
- Responsive design for mobile and desktop viewing.
- Showcases personal projects, skills, and contact information.
- Uses Vue.js for interactive UI components.
- SEO-friendly with `robots.txt` and web manifest configurations.

## Additional Notes
- The **assets/** folder contains static files such as logos and icons.
- The **images/** folder is for organizing portfolio images.
- Configuration files such as `.editorconfig` and `.prettierrc.json` help enforce coding standards.
- The **favicon** files and `site.webmanifest` ensure proper display on different devices.
- `robots.txt` is used for controlling search engine crawlers.
